== Description ==

Plugin adds a new section on setting menu to automatically add posts to any taxonomy. Wildcards are allowed meaning you could post something to every category you have with each post customized by the wildcard field.

I did this for a Real Estate client of mine, Deb Bennett who services the Petawawa area. Thought I would share it!

== Frequently Asked Questions ==

= How can I exclude unwanted term taxonomies for adding posts ?=

Add to Exclude Term field comma separated id of terms/categories that you don't want to be included in script

1. Upload automatic-posting to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
